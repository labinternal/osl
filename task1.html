Task 1: Practice UNIX Commands

1. cp Command (Copy)

Syntax: cp [options] source_file destination-file

Example: cp f1 f2

OUTPUT:
$ cat f1
This is GRIET
$ cat f2
This is GRIET

Options:

  • -f: Force copy 
    Example: cp -f f1 f2
$ cat f1
This is CSE
$ cat f2
This is GRIET
  • -i: Ask confirmation to overwrite
  • -b: Create backup before overriding

2. rm Command (Remove)

Syntax: rm [options] filename

Example: rm f1

OUTPUT:
f1 is deleted

Options:

  • -f: Ignore non-existing files 
    Example: rm -f myfile.txt
OUTPUT: Removes file myfile.txt
  • -r: Remove directory recursively
  • -i: Prompt before removal

3. mv Command (Move/Rename)

Syntax: mv [Option] source destination

Example:
$ cat > f1
hai
hello
cse
$ cat f2
hai
hello
$ mv f1 f2
$ cat f1
cat: f1: No such file or directory
$ cat f2
hai
hello
cse

4. chmod Command

Example:
$ chmod 777 f2
$ ls -l
total 1
-rwxrwxrwx 1 14 Apr  1 14:30 f2
-rw-r--r-- 1  0 Apr  1 14:18 hai

5. ps Command (Process Status)

Syntax: ps

PID TTY TIME CMD
644 01 10:30:00 bash
643 02 10:31:00 ps

6. kill Command

Syntax: kill <pid>

Example: kill 644

OUTPUT: The process gets terminated.
